# Logo da Lambari Kids

## AÇÃO NECESSÁRIA

Por favor, salve a imagem da logo oficial da Lambari que você enviou como:

**Nome do arquivo:** `lambari-logo.png`

**Caminho completo:**
```
d:\Projetos\Marco_Lambari(possivel cliente)\Lambari\public\images\brand\lambari-logo.png
```

## Instruções

1. Salve a imagem da logo (círculo azul com símbolo Lambari) nesta pasta
2. Certifique-se de que o nome do arquivo é exatamente `lambari-logo.png`
3. A logo deve ter fundo transparente (formato PNG)
4. Tamanho recomendado: pelo menos 500x500 pixels para qualidade

## Otimização (Opcional)

Para melhor performance, você pode:
- Comprimir a imagem em https://tinypng.com/
- Manter tamanho entre 50-100KB
- Garantir que a resolução é alta o suficiente para telas Retina

## Verificação

Após salvar a imagem, execute:
```bash
npm run dev
```

E verifique se a logo aparece corretamente no:
- Header (topo da página)
- Footer (rodapé)
- Favicon (aba do navegador)
